🚀 DevStart - Responsive Landing Page

A modern, responsive landing page built using HTML, CSS, and JavaScript.
This project is designed as an internship-level submission to showcase frontend development skills.

---

📌 Features

- ✅ Responsive design (works on mobile, tablet, desktop)
- ✅ Sticky navigation bar
- ✅ Mobile menu toggle (JavaScript)
- ✅ Smooth scrolling effect
- ✅ Clean and simple UI
- ✅ Contact form (demo functionality)

---

🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Media Queries)
- JavaScript (Vanilla JS)

---
